Data Pipeline Project Byte Example (Astraptes dataset)

I was not able to get my preferred dataset to upload to Google Fusion Tables - maybe they stopped developing this product?

So I made up a question to go with the data used in the tutorial - which herbivore species are associated with which host plant species in the region in which the data was gathered? The data on location and host plant species that were recorded to have a link with the observation of the species in the dataset.

I have displayed the results on the web page.

Source of data: "Astraptes fulgerator" Imported from astraptes_fulgerator_complex_sample_data (7).csv. Dan Janzen & Winnie Hallwachs - Edited on 2016 January 28
